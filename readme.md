<h1 align="center">Wejhtak PHP Application - MVC Based</h1>

Wejhtak is a PHP web application built from scratch using the MVC (Model-View-Controller) principle. It is a platform that allows students to apply for internships at various companies. Companies can approve or decline the applications, and the admin can assign supervisors to groups of students and send messages to the companies who approved the training of the students.

## Requirements

- PHP 8
- MySQL
- XAMPP

## Installation

1. **Clone this repository** to your local machine.
2. **Set up a local development environment** using XAMPP.
3. **Create a MySQL database** with the following configurations:
   - Database name: `mydb`
   - Collation: `utf8_general_ci`
4. **Import the database file** from `mydb.sql` located in the `database` folder.
5. **Start the XAMPP server** and navigate to `http://localhost/wejhtak/` to access the application.
